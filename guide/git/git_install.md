# Git Install

## Linux Install

1. Update the system and install git
    ```shell
    sudo apt update
    sudo apt install git git-lfs
    ```
2. Verify the install
    ```shell
    git --version
    ```
3. Configure the git settings
    ```shell
    git config --global user.name "Your Name"
    git config --global user.email "youremail@example.com"
    ```
5. Enjoy Git!
